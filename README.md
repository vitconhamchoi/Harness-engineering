# Harness Engineering

A practical repo for understanding and applying harness layers around AI coding agents.

## Read in this order
1. `README.md`
2. `01-FOUNDATIONS.md`
3. `02-INSTALLATION.md`
4. `03-HOW-TO-USE.md`

## What to copy into a real project
Only use files from:
- `project-kit/openspec/`
- `project-kit/local-superpower-patterns/`
- `project-kit/templates/`

## Important
- OpenSpec is a real external tool.
- `local-superpower-patterns/` is not an official Superpowers package/plugin.
- Do not copy this entire repo into a project. Only take what you need from `project-kit/`.
