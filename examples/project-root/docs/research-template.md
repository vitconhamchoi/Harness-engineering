# Research Template

## Purpose
Use this template when the task requires searching, comparing sources, and returning a reliable conclusion.

## Research Rules
- Search before answering when fresh data matters.
- Prefer primary or near-primary sources.
- If sources disagree, say so clearly.
- Distinguish facts, inference, and opinion.

## Recommended Structure
1. Summary
2. Key evidence
3. Conflicts or uncertainty
4. Practical conclusion

## Good Use Cases
- live market checks
- product / pricing comparisons
- security advisories
- technical feature verification
- company / tooling due diligence

## Anti-patterns
- answering from stale memory when live lookup is easy
- presenting one source as certain when others disagree
- copying large unverified text blobs into the answer
