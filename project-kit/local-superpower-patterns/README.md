# Local Superpower Patterns

Phần này **không phải official Superpowers package/plugin**.

Đây chỉ là các pattern local để mô phỏng tinh thần:
- brainstorming trước khi code
- viết plan nhỏ, rõ, verify được
- chia task cho sub-agent khi cần
- TDD và verification trước delivery
- worktree discipline để không phá working tree chính

## Khi nào dùng
Dùng khi mày muốn:
- đưa workflow kỷ luật hơn vào project
- nhưng chưa tích hợp hoặc chưa cài official Superpowers implementation

## Khi nào KHÔNG nên hiểu nhầm
Thư mục này không có nghĩa là:
- mày đã cài package bên ngoài
- mày đã có slash commands thật
- mày đã có plugin chính thức cho Cursor/Claude/Codex/OpenCode
