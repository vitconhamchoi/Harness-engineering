# What is Superpowers?

Theo framing của repo này, Superpowers là lớp design + execution skills như:
- brainstorming
- writing-plans
- TDD
- sub-agent execution
- git worktree discipline

## Quan trọng
Repo này hiện KHÔNG cung cấp official Superpowers package/plugin.

Thư mục `project-kit/local-superpower-patterns/` chỉ là pattern local để mô phỏng tinh thần đó.

Nếu cần install path chính thức của một implementation bên ngoài, phải verify riêng theo tool/repo gốc.
